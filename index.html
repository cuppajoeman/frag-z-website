most points win (mpw)

this is primary game mode being developed, this is a novel game mode where all the players have infinite ammo, start with all weapons in the game and a time limit for 10 minutes commences. all players start with 0 points and points are scored by successfully hitting enemies with the weapons, the player with the most points at the end of the match wins the game.

points

if you shoot an enemy and gain X points from the shot, then the enemy loses X points, each weapon has a base amount of points earned for each
